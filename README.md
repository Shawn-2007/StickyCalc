# ⚙️ 功能簡介

這是一款結合便利貼、節點、計算機與圖表的網頁工具，主要功能如下：

- 📝 會員註冊與登入功能  
- ☁️ 雲端儲存與載入便利貼與版面資訊  
- 🧾 便條紙文章紀錄  
- 🔗 節點連線計算功能  
- 📊 圖表展示 (ApexCharts)  
- 💰 記帳與資金管理
